# Overdue Portal (Next.js)

## 1) Setup
- copy `.env.example` to `.env.local`
- set your values

## 2) Install and run
```bash
npm install
npm run dev
```

Open:
- http://localhost:3000/overdue?token=YOUR_TOKEN

Optional limit:
- http://localhost:3000/overdue?token=YOUR_TOKEN&limit=2000

## Interakt button link
Use your deployed URL in template button, for example:
`https://your-domain.com/overdue?token=YOUR_TOKEN`
